# Google Omni Video Editing Prompts

Source: Google AI Studio (aistudio.google.com) → Video Generation → Omni Model

## Setup
1. Sign in with Google account
2. New project → Video Generation → Select Omni Model
3. Upload footage (selfie, talking-head, product, or B-roll)

---

## Core Editing Prompts

### Background Replacement
```
Replace the entire background with a luxury penthouse overlooking Dubai, 
while preserving the subject and maintaining realistic lighting.
```

### Location Transformation
```
Transform the location to a tropical beach while keeping the subject's 
clothing and maintaining the same camera angle and lighting continuity.
```

### Cinematic Lighting
```
Convert the lighting into cinematic golden hour lighting with warm 
tones and realistic reflections on the subject's face and clothing.
```

### Object Removal
```
Remove all background distractions and clutter while keeping the 
main subject completely intact with natural edge blending.
```

### Object Addition
```
Add a [vehicle/object] to the scene that matches the perspective, 
lighting conditions, and overall visual style of the footage.
```

### Environment Swap (Ultra Style)
```
Place the subject bungee jumping off a bridge in New Zealand, 
match the lighting to the original footage, keep facial features 
identical, dynamic camera movement tracking the fall.
```

## AI Avatar Creation
1. Upload 30–60 seconds of clean speaking footage
2. Supply 2–3 reference photos (frontal + side angles)
3. Prompt:
```
Create a digital avatar preserving this person's exact facial 
features, hairstyle, skin tone, and natural expressions for 
use in future AI-generated video content.
```

## Viral Content Formulas for Reels/Shorts

### MrBeast Format
```
Fast-paced sequence: open with subject doing [unexpected action], 
cut to 3 rapid reactions at 0.5x speed, close with [reveal moment]. 
Dynamic camera movement, high saturation, sharp contrast.
```

### Sports in Unexpected Setting
```
Show [subject] playing [sport] in [impossible location — Eiffel Tower 
rooftop, underwater, space station]. Match physics and lighting to 
feel photorealistic. 8–10 second clip, single camera angle.
```

---

## Workflow
Record → Upload to AI Studio → Background/Lighting/Object edits → 
Avatar consistency check → Style transfer → Export → Post to Reels/Shorts

## Best Use Cases
- Creator reels (Instagram Reels, YouTube Shorts, TikTok)
- Product ads and e-commerce demos
- Personal brand / education content  
- Agency UGC generation

## Key Insight
Use Omni for **efficiency** not replacement — AI removes location 
and lighting constraints, human provides storytelling and energy.
