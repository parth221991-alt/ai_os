# F03 — HOLLYWOOD VFX

**Formula ID:** F03  
**Type:** Output-First Visual Showcase  
**Reel Duration:** 20–30 seconds  
**Production Complexity:** High — requires high-quality AI-generated visual asset as source material  
**Repeat Frequency:** Medium — use 1–2× per week when strong visual outputs are available  
**Primary CTA:** DM trigger (TOOLS) or Save CTA  
**Derived from:** APIFY Week 3 theme "PROOF — Here's what's possible" + AI_SNIPP_REEL.md lesson "Premium UI visuals outperform generic AI animations"

---

## Purpose

Show what AI can generate that looks indistinguishable from professional production. The formula leads with the output — the viewer sees the result before they know it's AI-generated. The reveal that "this was made by AI + one prompt" is the core emotional beat.

**Why it works:** Visual proof is the most shareable content format in the AI niche. When someone sees a genuinely impressive AI output, their first instinct is to share it with someone who needs to see it. This formula is optimized for shares and saves over comments.

**Psychological mechanism:** Aesthetic pleasure → cognitive dissonance ("this can't be AI...") → reveal → amazement → share it.

**Production requirement:** This formula only works if the AI output is genuinely impressive. A mediocre Google Flow clip with this formula is worse than no clip at all — it actively damages credibility. Only deploy when you have a strong output.

---

## Best Use Cases

- Google Flow generated cinematic scenes (film-quality shots)
- Midjourney/Ideogram photorealistic image generation
- Kling AI video outputs
- AI-generated music + video combinations
- AI-powered video editing results (input video → transformed output)
- Code-generated visual interfaces (dashboard built by Claude in one conversation)
- AI-written content at a quality that's genuinely surprising

**Avoid for:** Average or mediocre outputs. Stylized/animated outputs (saves these for a different formula). Anything that looks clearly AI-generated in a way that reduces rather than amplifies amazement.

## Hook Structures

**Hook Type A — Output First (strongest — creator doesn't appear first)**
```
[0-3s: Show the AI output with zero context. No creator. No text. 
Just the cinematic frame, the photorealistic image, the transformed video.]
[3s: Hard cut to creator]
Creator: "Ye ek prompt se bana hai."
```

**Hook Type B — Disbelief Setup**
```
"Main tumhe kuch dikhata hoon — 
aur phir batata hoon kaise bana.
Pehle dekho."
[Cut to the output]
```

**Hook Type C — Context Challenge**
```
"Agar main tumhe ye batata hoon ki ye 
Hollywood production se nahi hai — 
balki AI se bana hai — tumhe believe hoga?"
```

**Hook Type D — Time Reveal**
```
"Ye banane mein mujhe lagey the — 
[pause] — 8 second."
[Show the output]
```

**Selection rule:** Type A is the gold standard when the output speaks for itself. Type B/C when the output needs framing to be fully appreciated. Type D for time-efficiency content (pairs well with F06 AI vs Human).

## Story Structure

```
[0-3s]   COLD OPEN   → Show the output. No creator. No context. Pure aesthetic impact.
[3-6s]   REACTION    → Creator reacts. Close-up face. "Ye ek prompt se bana hai."
[6-18s]  REVEAL      → Show the tool, show the prompt, show the generation process briefly.
[18-25s] PROOF       → Show the output again, this time in context. Comparison if possible.
[25-30s] CTA         → "Save karo" or "DM karo TOOLS for the prompt."
```

**The cold open is what separates this from all other formulas.** No other AI_SNIPP formula opens without the creator. This visual-first approach is a pattern interrupt at the feed level — when someone is scrolling, they see a cinematic image before they see the creator. This catches scroll-stoppers who wouldn't have stopped for a creator-first opening.

## Character Usage

**Clip 0:** Creator not present. The output IS the character of this clip.

**Clip 1:** Extreme close-up (from character_consistency_guide.md Block C — Reveal/Reaction). CONTROLLED AMAZEMENT direction: "Eyes slightly widened. A slow exhale through slightly parted lips. One eyebrow rises fractionally." The glasses catch a faint reflection of the output. Zero dialogue for first 2 seconds. At second 2: slow turn to camera. At second 3: delivers the line. The restraint is critical — over-performing here destroys the credibility the output just built.

**Clip 2:** Desk setup, facecam or full-frame tool interface. "The focused professional showing the process" energy. Not amazed anymore — this is the explanation mode. Matter-of-fact.

**Clip 3:** Facecam overlay, bottom-right. Creator nods slowly while output is shown. The "see what I mean?" expression from Clip 1 has resolved into "exactly what I expected."

**Clip 4:** Warm CTA close. Eye level. Genuine. Save CTA or DM trigger for the prompt.

## Editing Style

- **Cold Open treatment:** The output clip plays with NO graphics, NO captions, NO music for the first 1.5 seconds. Then music fades in softly (0.5s fade). No creator lower-third. No "@ai_snipp" watermark during these 3 seconds (it appears from Clip 1 onward).
- **The cut from Cold Open to Reaction:** Hard cut. No transition. The abruptness is the impact.
- **During the Tool Reveal:** Show the actual prompt in text on screen — typed or static. This is the most valuable visual element in the reel. It must be legible. Use a dark-mode terminal or AI interface. The prompt text appears as a brief text highlight.
- **Output comparison treatment (if used):** Side-by-side or before/after wipe. The traditional Hollywood production reference vs the AI output.
- **Color grade:** For the Cold Open and final Output display, enhance the AI output's color: push contrast, add cinematic warmth or coolness depending on the content. Make it look as good as it can.

---

## Audio Style

- **Cold Open (0-1.5s):** SILENCE. Then music fades in softly.
- **Music:** Cinematic, atmospheric — not upbeat electronic. This formula is about aesthetics. The music should match the AI output's visual tone. A dramatic space scene needs cinematic orchestral. A photorealistic portrait needs a warm ambient.
- **Creator voice:** When creator speaks (Clip 1 onward), music drops to 20% volume. Creator voice is primary.
- **SFX:** A soft "impact" or "whoosh" on the cut from Cold Open to creator. A keyboard typing sound when the prompt appears on screen. A subtle riser before the final output reveal.
- **No SFX on the output itself** — let the visuals breathe without audio interruption.

## Flow Generation Strategy

**Critical pre-production requirement:** The AI output (Clip 0) must be generated and evaluated BEFORE any creator clips are generated. Do not invest in creator clip generation if the output isn't strong enough. The output quality sets the ceiling for this formula.

**Clip 0:** Generate using appropriate tool (Google Flow, Kling, Sora, Midjourney). For Google Flow:
- Use a separate Flow session dedicated to the output
- Do not include creator character in this session
- Optimize for visual quality above all else
- Test multiple variations before selecting the hero output

**Clip 1 (Reaction):** Block 7 — REACTION/REVEAL. Extreme close-up. CONTROLLED AMAZEMENT. Static camera. Key addition to prompt: "The glasses catch a faint warm reflection from the left side, as if a bright, beautiful screen is illuminating the character's face from slightly off-frame. This ambient light is subtle — it suggests what he is watching without showing it."

**Clip 2 (Tool Demo):** Block 3 — TOOL DEMO Phase 1. Desk setup. Show the tool interface and the prompt. The tool UI must be a recognizable, realistic interface — not generic.

**Clip 3 (Facecam + Output):** Block 2 — FACECAM OVERLAY. Main content area: the AI output again, but this time it may have a light graphic overlay identifying the tool. Creator facecam bottom-right. Creator expression: the satisfied nod.

**Clip 4 (CTA):** Block 5 — CTA. Standard warm close. DM trigger for prompt.
