## CONTENT EXTRACTION WORKFLOW

*A structured process to extract actionable intelligence from competitor content. Run weekly on highest-performing competitor posts.*

### Step 1 — Post Identification (5 min)

1. Open each competitor account
2. Sort by recent posts (last 7 days)
3. Identify posts with engagement rate significantly above their account average
4. Save URLs of top 2–3 posts per competitor to the weekly trend log

**Flag for extraction:** Any post with engagement rate > 2× account average within 24 hours of posting.

---

### Step 2 — Hook Extraction (10 min)

For each flagged post:

1. **If Reel:** Watch with sound off. What visual element made you stop? Note it.
2. Watch with sound. Write down the exact first 10 words spoken.
3. Ask: "What emotion did this first line trigger?" (curiosity / fear / excitement / recognition)
4. Ask: "What pattern does this hook belong to?" (refer to hook_library.md categories)
5. Draft @ai_snipp Hinglish adaptation of the hook concept (not word-for-word — adapt the pattern)

**Output format:**
```
Original: "[exact first line]" — @[handle]
Pattern: [Hook category from hook_library.md]
Emotion triggered: [curiosity / fear / excitement / recognition / shame]
@ai_snipp adaptation: "[Hinglish version in creator's voice]"
Formula candidate: F[NN]
```

---

### Step 3 — Comment Bucket Mining (15 min)

For top 2 posts per competitor, read first 50 comments. Classify each into:

**BUCKET A — Questions** (comments ending in "?" or starting with "how", "what", "where", "when", "why", "can you")
> Content signal: This is a topic the audience doesn't understand yet — educational opportunity.

**BUCKET B — Requests** ("can you make a video on", "next video should be", "please cover", "I'd love to see")
> Content signal: Explicit demand for content that hasn't been made — first-mover opportunity.

**BUCKET C — Frustrations** (contains "wish", "confusing", "doesn't work", "nobody explains", "I've been trying", "so hard to")
> Content signal: Real pain language. Extract exact phrases for hook_library.md.

**BUCKET D — Praise** ("love how you", "finally someone who", "this is exactly what I needed", "the only creator who")
> Content signal: What the audience values most about this creator — what is @ai_snipp missing?

**Output format:**
```
COMMENT BUCKET MINING — @[handle] — [Post title/topic]

BUCKET A (Questions):
  Q1: "[exact comment]"
  Q2: "[exact comment]"
  Content opportunity: [one reel idea]
  Hook potential: [draft hook using this question language]

BUCKET B (Requests):
  R1: "[exact comment]"
  Content opportunity: [one reel idea — is this a gap @ai_snipp can fill?]

BUCKET C (Frustrations):
  F1: "[exact comment]" ← HIGH PRIORITY — real pain language for hooks
  F2: "[exact comment]"
  Hook phrase extracted: "[Hinglish adaptation]"

BUCKET D (Praise):
  P1: "[exact comment]"
  What they're praising: [specific element]
  @ai_snipp gap: [does @ai_snipp deliver this? Yes/No/Partially]
```

---

### Step 4 — Cross-Competitor Synthesis (10 min, weekly)

After mining all 5 competitors weekly, answer:

1. **Which topic appeared across 3+ competitor comment sections as a request or question this week?**
   → Validated demand: produce this week.

2. **Which hook formula appeared in the top-performing post for 2+ competitors this week?**
   → Proven hook: adapt for @ai_snipp's next reel.

3. **What does the audience complain about that NONE of the 5 competitors address well?**
   → First-mover gap: design a reel specifically for this gap.

4. **What was the fastest-growing post format this week across all 5 competitors?**
   → Format signal: consider adopting this format if not already in the mix.

**Output:**
```
WEEKLY CROSS-COMPETITOR SYNTHESIS — [YYYY-MM-DD]

Top validated demand this week: [topic]
  Found in: @[competitor1], @[competitor2], @[competitor3] comment sections
  @ai_snipp content idea: [reel concept]
  Formula: F[NN] | Priority: High

Top hook formula this week: [hook pattern name from hook_library.md]
  Example: "[exact hook from competitor's top post]"
  @ai_snipp adaptation: "[Hinglish version]"

First-mover gap identified: [topic nobody addresses well]
  Evidence: [which comment buckets from which accounts]
  @ai_snipp opportunity: [reel concept]
  Formula: F[NN]

Fast-growing format this week: [format]
  Account: @[handle] | Engagement: [N]
  Applicable to @ai_snipp: Yes / No / Partially
  If yes — next implementation: [when + which formula]
```
