# AI_SNIPP Pipeline — Background Music

Drop your background music file here as `background.mp3`.

## How to Generate (Free)

**Suno.com (free tier — 10 songs/day):**
Use this prompt:
```
Confident forward-moving electronic instrumental, medium-fast tempo, no lyrics,
60 seconds, builds slightly over time, warm and energetic, creator studio vibe
```

Download as MP3. Rename to `background.mp3`. Drop it in this folder.

## Usage in Pipeline

The pipeline mixes this file at 25% volume under the clip audio.
The music fades out 3 seconds before the end of the reel.
Speech always stays primary — the music is subtle, not competing.

## Requirements

- Format: MP3 (preferred) or WAV, M4A, AAC
- Duration: at least as long as your longest reel (typically 35+ seconds)
- No vocals — instrumentals only (avoids Instagram copyright detection)
- Copyright-free or generated (Suno output is royalty-free for personal use)

## Without This File

The pipeline proceeds without background music and outputs a silent-background video.
You can add music manually in CapCut after the pipeline finishes.
