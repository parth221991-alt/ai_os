### R1 — TOPIC

**Working Title:** Claude 4 Launch — Indian Developers Ko Kya Milta Hai

**Core Value Claim:** Anthropic's Claude 4 family (Sonnet 4.6, Opus 4.7) brings specific capability upgrades that directly affect how Indian developers and creators should use AI in their workflow.

**Viewer Pain Point:** "There are new AI models releasing every few weeks. I don't know which ones actually matter for my work and which are just hype. I need someone to filter this for me."

**Time-sensitive:** Yes — produce within 24 hours of announcement. Window: 48 hours before this reel becomes stale.

**Repeat check:** No prior reel on Claude 4 specifically.

### R2 — FORMULA SELECTION

**Selected:** F07 — News Flash ✓

**Selection justification:** Real AI development, <48 hours old, Indian relevance angle (developers, creators using Claude). Exactly F07's "Best for" case — translating a model release into plain Hinglish implications for non-researcher audience.

**2nd-best alternative:** F02 (Future Shock) — loses because this is a news update, not a threat/urgency play. F07 is appropriate for "here's what happened and what it means" framing. F02 would be appropriate only if the capability gap was framed as a career threat.

### R3 — HOOK

**Hook Type:** F07-A — Breaking Announcement (most urgent)

**Raw Hook Line:**
> "Breaking: Anthropic ne Claude 4 launch kiya. Yeh 5 changes tumhare kaam mein seedha asar dalenge."

**Hook Source:** Real announcement — Claude Sonnet 4.6 and Opus 4.7 releases. Indian developer and creator audience is a primary Claude user segment.

**1-second scroll test:** Passes — "Breaking:" pattern interrupts scroll immediately. "5 changes" creates enumeration curiosity.

### R4 — CTA INTENT

**Primary CTA:** Follow for more updates

**CTA Exact Line:**
> "Aise AI updates ke liye — follow karo @ai_snipp. Main filter karta hun jo tumhare kaam ka hai."

**Lead magnet:** None (Follow CTA — no DM trigger needed)

### R5 — PROOF POINTS

1. Claude Sonnet 4.6 — primary model for reasoning and code work, same model powering Claude Code CLI
2. Claude Opus 4.7 — highest-capability model, suited for most complex multi-step reasoning
3. Claude Haiku 4.5 — high-throughput tasks (classification, real-time UI feedback) at low cost

**Source:** Anthropic model documentation, June 2026
**Reproducible:** Yes — anyone can access Claude.ai or the Anthropic API
